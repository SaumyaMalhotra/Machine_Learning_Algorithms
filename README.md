Machine Learning Algorithms – Scikit-Learn Practice
Overview

This repository contains hands-on practice notebooks focused on machine learning algorithms using Scikit-Learn. The work is based on small, real-world datasets and is aimed at understanding how different ML models work, how to train them, and how to evaluate their performance.

Files & Description

Machine Learning Algos.ipynb
Practice notebook covering core machine learning algorithms such as:

Linear Regression

Logistic Regression

Decision Trees

Random Forest

Model evaluation basics

Scikit_lib.ipynb
Practice notebook focused on using the Scikit-Learn library, including:

Data preprocessing

Model training and prediction

Train-test split

Model saving and loading

Datasets Used

income.csv – income prediction

titanic.csv – survival prediction

salaries.csv, hiring.csv – salary prediction problems

HR.csv – employee attrition analysis

homeprices.csv, area_price.csv, canada_pci.csv – regression examples

carprices.csv, carprices1.csv – car price prediction

price.csv, price.xlsx – pricing-related analysis

test_scores.csv – model testing and evaluation

model_pickle
Saved machine learning model using Python pickle for reuse and deployment basics.
